# DBM.functions

This package contains a variety of functions for a variety of purposes.

It includes a function, `Aligning_Values_Across_Multiple_Vertical_Axes`, that generates axis limits for aligning values across multiple vertical axes on plots.

It includes a function, `Comparing_Multiple_Independent_Correlation_Coefficients`, that compares 3 or more correlation coefficients from independent correlations (Levy, 1977).

<b>Works Cited</b>

Levy, K.J. 1977. Pairwise comparisons involving unequal sample sizes associated with correlations, proportions or variances. Br. J. Math. Stat. Psychol. 30:137-139.
